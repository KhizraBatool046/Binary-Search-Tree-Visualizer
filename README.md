# Binary Search Tree Visualizer

## Project Overview

This project implements a Binary Search Tree (BST) as part of the **Data Structures and Algorithms (DSA)** course during the third semester of my BS Computer Science degree.

The project demonstrates the implementation of BST operations and helps in understanding how binary search trees store, search, and manage data efficiently.
